Machine learning, more than anything else, pushes at the boundaries that we draw between computers and humans. If a computer might one day write stories like we do, will they really be that different from us? WILL a computer ever be able to?

Create explores these questions and wades into the realm of machine learning and artificial intelligence. Here, we have adapted what is known as a recurrent neural network (RNN), a type of artificial computing network that is able to process input (such as words, pictures, patterns) and then generate similar output. Not only so, what makes RNNs so special is that every time they generate an output, that output gets added to its collection of inputs. In this way, the longer a RNN processes information, the more intelligent and accurate it gets.

You can see why this might have applications for literature. Given any story as an input, we can get the computer to generate repeated iterations of text as output, based off the patterns of words and speech the computer can glean. Each iteration is fed back to the computer as input, hence, the computer might start off writing gibberish, but the longer it processes, the closer the computer gets to the actual text.
